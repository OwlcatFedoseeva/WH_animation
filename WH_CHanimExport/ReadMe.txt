Описание Скрипта

Этот скрипт экспортирует анимации как Human, Eldar, SpaceMarine так и Кричей.

У Human и Eldar запекаются только ротейты на основном скелете, юбка и кости оружий в руках запекаются и по транслейтам.
У SpaceMarine список запекаемых косточек немного отличается, помимо юбки и оружия, по транслейтам запекаются кости ног и кости наплечников.

Добавлена кнопка для экспорта Кричей.

Как пользоваться.

Если экспортируем Персонажей
- Открыть файл анимации для экспорта.
- Нажав кнопку Set Path задаем папку куда мы хотим сохранить анимацию
- Если это анимация Human или Eldar, во вкладке оставляем/выбираем Human
- Жмем кнопку Export Character Animation

- Если это анимация SpaceMarine, во вкладке выбираем SpaceMarine
- Жмем кнопку Export Character Animation


Если экспортируем Кричей

- Открыть файл анимации для экспорта.
- Нажав кнопку Set Path задаем папку куда мы хотим сохранить анимацию
- Жмем кнопку Export Creature Animation


Установка

1 - сохранить всю папку [WH_CHanimExport] в: C:\Users\YOURUSERNAME\Documents\maya\scripts\

2 - Из папки [WH_CHanimExport] перетащить файл install.py в открытую Maya 

3 - На полке должна появиться зеленая иконка  Expot Anim
